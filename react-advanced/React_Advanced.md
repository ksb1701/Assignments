# React Advanced

Welcome to the React Advanced module! This folder contains all the assignments and exercises designed to take you beyond the basics of React.

## Module Structure

We are breaking down advanced React concepts into manageable, hands-on sessions. Each session has its own folder containing the code and a dedicated `README` / `.md` file explaining the tasks.

## Getting Started

1. Make sure you are in the `react-advanced` directory in your terminal.
2. Run `npm install` if you haven't already.
3. Run `npm run dev` to start the local development server.
4. Open `src/App.tsx` and import the component you wish to view from the respective session folder.
